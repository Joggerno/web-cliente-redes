# Sitio web Grupo Redes S.A.

Proyecto base para una landing page corporativa sencilla y moderna.

## Cómo abrirlo en VS Code

1. Descomprime este ZIP.
2. Abre VS Code.
3. Ve a `File > Open Folder`.
4. Selecciona la carpeta `grupo-redes-site`.
5. Abre el archivo `index.html`.
6. Haz clic derecho y selecciona `Open with Live Server`.

## Extensiones recomendadas para VS Code

- Live Server
- Prettier - Code formatter
- Auto Rename Tag
- HTML CSS Support
- Path Intellisense

## Archivos principales

- `index.html`: estructura del sitio.
- `css/styles.css`: estilos visuales y diseño responsive.
- `js/main.js`: menú móvil, filtros de proyectos y conexión del formulario a WhatsApp.
- `assets/img`: imágenes optimizadas para la web.

## Pendientes antes de publicar

- Reemplazar teléfono, correo y enlace de WhatsApp.
- Confirmar logo oficial en buena calidad.
- Confirmar autorización para usar fotos, logos visibles de terceros y nombres de clientes.
- Revisar fotos con coordenadas, mapas, placas o datos sensibles antes de publicar.
- Cambiar textos según la versión final aprobada por el cliente.
